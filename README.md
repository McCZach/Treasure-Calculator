# Advanced DnD (1E) - Treasure Generator
## Treasure from DMG - Random Determination Tables
Tkinter GUI application that implements the Treasure (Random Determination) tables

Tables Implemented: <br>
```
1. POTIONS
2. RINGS
3. SCROLLS
4. RODS, STAVES, & WANDS
5. ARMOR AND SHIELD
6. SWORDS
```
